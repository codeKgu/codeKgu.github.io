SEASide

Inspired by our personal struggles in introductory computer science classes, SEASide is a cross platform graphical user interface that allows users to login to UCLA's SEASNET linux servers. SEASide creates tabs for each connection to the server and uses custom server settings to automate the process. For each connection a file explorer is displayed to streamline navigation on the server. In addition, a terminal is emulated that securely streams commands entered by the user to the server.

Tools Used: Python, paramiko, kivy, ssh, sftp
